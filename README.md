# Mini Siri – AI Desktop Voice Assistant

Mini Siri is a Python-based AI desktop assistant that accepts voice and text commands, generates AI responses, opens applications, searches the web, manages clipboard content, and performs basic Windows automation.

## Features

- Voice-to-text commands
- Text-to-speech responses
- Gemini AI integration
- Open Spotify, browser, VS Code, Discord and other apps
- Google and YouTube search
- Generate emails, notes and explanations
- Copy responses to clipboard
- Open common Windows folders
- Take screenshots
- Pastel desktop interface
- Preparing, Listening, Thinking and Speaking states

## Technologies Used

- Python
- CustomTkinter
- SpeechRecognition
- pyttsx3
- Gemini API
- PyAutoGUI
- Pyperclip
